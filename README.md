## 🚀 Quick Start

```bash
# Clone the magic ✨
git clone https://github.com/JSXRakibulhasan10/ScoreSpare.git

# Enter the arena ⚽
cd ScoreSpare

# Install dependencies 📦
npm install

# Launch the app 🚀
npm start
```

### Environment Setup
```env
# Backend Configuration
PORT=5000
FOOTBALL_API_KEY=your_api_key_here
CHATBOT_API_KEY=your_chatbot_api_key
```
